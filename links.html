Ox
The programs are written in Ox, which is free for academic use (but not open source). The non-free Ox Professional version is installed on the virtual machine. Ox is available for Windows, Macs and Linux.
CGI and HTML forms
The prototype uses an initial version of CGI parser for Ox so that (eventually) the code will be self-contained, producing HTML input forms and parsing the posted data and running the model.
