Objective

The overall goal of this project is to create an interactive website that allows users to explore the long-term implications of climate change and economic policy changes in response to climate change.

Specifically, the project seeks to implement Leach (2008) which presents a stylized model of the world economy that affects and is affected by climate change through greenhouse gas emissions and temperature impacts on productivity. The effect of alternative economic policy responses on welfare (in the technical economic sense) is considered.

The strategy is:
  • Update Leach's code that solves the model, then extend it to account for other policies.
  • Create a web interface that allows for remote setting of parameters of policy experiments
  • Integrate updated temperature and economic data used in the analysis.
  • Add other integrated assessment models.
